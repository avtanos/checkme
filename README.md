# checkme

Веб-приложение для поиска провайдеров услуг (грузовые машины, сантехники, эвакуаторы, электрики) на карте Кыргызстана.

## Технологии

- **Backend**: FastAPI (Python)
- **Frontend**: React
- **База данных**: SQLite
- **Карты**: Leaflet

## Установка и запуск

### Backend

```bash
cd backend
pip install -r requirements.txt
python init_data.py  # Инициализация тестовых данных
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm start
```

Приложение будет доступно по адресу `http://localhost:3000`

## Функциональность

- Карта с маркерами провайдеров услуг
- Фильтрация по категориям
- Регистрация и вход для провайдеров
- Личный кабинет провайдера для управления данными
- Загрузка фото
- Выбор местоположения на карте
- Интеграция с WhatsApp для связи с провайдерами
